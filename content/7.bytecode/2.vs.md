# Bytecode vs Interpreter
| Bytecode | Interpreter |
|----------|-------------|
| Faster | Slightly Slower |
| Ahead of time | Just in time |
| Harder to read | Easier to read |
| Harder to write | Easier to write |
| Cross platform | Cross platform |
| Harder to debug | Easier to debug |
| Smaller file size | Larger file size |
| Harder to decompile | Easier to decompile |
