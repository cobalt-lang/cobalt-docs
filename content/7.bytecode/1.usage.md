# Bytecode Usage
## Usage
```
cobaltc <file.cobalt>
```
this will result in a new file, the bytecode file.
You can run the bytecode file with the interpreter:
```
cobalt <file.cobalt>
```
## Lua -> Cobalt Bytecode
Sadly, there is no way to convert Lua to Cobalt Bytecode. But 
you can compile Lua code to Cobalt but not Bytecode to Bytecode.
