# Introduction
Cobalt is based of Lua `5.4.3` and has been modified to stay up to date with
`5.4.6`.

This will give a base guide on what all has been changed or added.

***
## Preprocessor
Preprocessor written 100% in Cobalt (`#define`, `#open`, `#include`, `#ifdef`, `#ifndef`, `#endif`, `#else`, `#elseif`)
## Type system
Type annotations and a type checker (in preprocessor) is available.
## Syntax
Improved syntax based on LJS
### Lua:
```lua
local x = 10
for i = 1, x do
    print("Hello!")
end
```
### Cobalt:
```js
var x = 10;
for (i = 1, x){
    print("Hello!");
}
```

## Real threads
Real CPU threads for Windows and Unix systems:
```lua
thread = async.new([[
    print("Hi")
]])
thread->start()
```
## New libraries
`core`, `device`, `struct`, `complex`, `Vector3`, `Vector2`, `Color`, `Transform`, `signal`, `file`, `msg`, and more libraries written for cobalt or ported from C or Lua.
## Arrays
A subset of tables called arrays are available which do not have indexs.
## Performance
- AOT (2x)
- For Loops (3x)
- Interpreter Pool Allocator (30%)
## Time methods
`wait` (second), `swait` (second), `mwait` (millisecond), `umwait` (microsecond)

## `warn`, `info`, `abort`
`warn`, `info`, and `abort` are available.
### `warn`
Warn outputs a warning like:
> **warning:** Text here
### `info`
Info outputs a warning like:
> **info:** Text here
### `abort`
Abort ends the process as fast as possible without performing an memory cleanup and ending the interpreter as fast as possible.

## `$` & `@`
`$` is an alias for `pairs` and `@` is an alias for `ipairs`.

```js
for (i, v in $tbl){ // Works (with preprocessor)
    print(i,v)
}
```

## `~`
`~` is equivelent to Python Walrus and is calculated on preprocess time. 

```lua
if (x~ math.random(1,10) > 10){
    print(f"{x} is greater than 10")
}else{
    print(f"{x} is less than 10")
}
```

## `_` 
`_` is a library full of system/language variables such as:
```js
_.VERSION // -> 2300 (cobalt23)
_.LUA // -> 506 (Lua 5.4.6)
_.global("x", 1) // returns 1
x // 1
```

## Binary and Octal numbers
```js
var binary = 0b1101 // -> 13
var octal = 0o15 // -> 13
var hex = 0xD // -> 13
var number = 13  // -> 13

print( (number == octal) && (number == binary) && (number == hex) )
```

## `<pre>` values
```js
var x <pre> = ("Hello, ".."world".."!") 
// pre values allow you to rather than on runtime concat the values
// but rather on preprocess time. The generated code would be:

var x = "Hello, world!"
```

## `&`

```js
var tbl = {1,2,3,4}
var item1, item2 = &tbl

item1 //-> 1
item2 //-> 2

func(&tbl)

/* Equivelent to */

var tbl = {1,2,3,4}
var item1, item2 = table.unpack(tbl) // or 1, 2

item1 //-> 1
item2 //-> 2


func(1,2,3,4)
```

`&` turns tables into tuples, useful when providing arguments or fetching items from a table.
