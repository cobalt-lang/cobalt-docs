# Increment Operators
Lets say you want to add 1 to a variable, In Lua you would do
```lua
local variable = 10
variable = variable + 1
```
Using compound operators in Cobalt you would use 
```js
var variable = 10;
variable =+ 1;
```
But using Increment operators it is just
```js
var variable = 10;
variable++;
```
***
### `++`
+1 a variable
```js
var variable = 10;
variable++; // 11
```
### `--`
-1 a variable
```js
var variable = 10;
variable++; // 9
```
