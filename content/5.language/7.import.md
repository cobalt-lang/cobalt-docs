# Import
You can import libraries using the `import` or `require` keywords.
    
```js
// import
helpers = import("helpers")

// require
helpers = require("helpers")
```

## Importing pre-compiled libraries
You can import .so or .dll files using the `import` keyword.
    
```js
// NOTICE: helpers.so is not a real file, it's just an example

// import
helpers = import("helpers.so") // Unix/Posix
helpers = import("helpers.dll") // Windows

// require
helpers = require("helpers.so") // Unix/Posix
helpers = require("helpers.dll") // Windows
```

## Returning values in a library
```js
// Inside of library
var module = {}

return module // Returns the module to who ever required it
```
