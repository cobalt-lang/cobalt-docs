#title
New Features

#root
:ellipsis{left=0px width=40rem top=10rem blur=140px}

#default
   ::card{}
   #title
   Portability
   #description
   We are going to add more platform support, like WebAssembly, Android, Swift bridges, and more.
   ::

   ::card{}
   #title
   Performance
   #description
   cobalt24 will introduce many performance improvements, including a better JIT compiler, a new garbage collector, and more.
   ::

   ::card{}
   #title
   Extended Standard Library
   #description
   cobalt24 will introduce a larger standard library.
   ::

   ::card{}
   #title 
   Decorators
   #description
   Decorators will be introduced allowing you to use `@` to add metatables, 
   attributes, or bind to a signal.
   ::

   ::card{}
   #title
   Better Types
   #description
   A typechecker will be added so if the type doesnt follow the annotation rule it will not change, and `!` will make it error, and tables like 
   `[string, !number]` will be supported. Custom types can also be defined.
   ::

   ::card{}
   #title
   Pointers
   #description
   Variable Refrences, Pointers will be added allowing a new variable to be
   created that refrence to the original.
   ::

   ::card{}
   #title
   Better Output
   #description
   cobalt24 will have better traceback, more warnings, and a new library for handling gradients, colors, and positioning in the output. 
   ::
   
   ::card{}
   #title
   LLVM
   #description
   cobalt24 will experiment with using LLVM optionally for JIT and AOT.
   ::

   ::card{}
   #title
   JIT
   #description
   A JIT compiler will be available based on DynASM.
   ::
 ::
