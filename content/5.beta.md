New Features

#root

#default
  ::card{}
  #title
  Import C/++ Files
  #description
  cobalt24 will allow you to import C and C++ files, which will allow you to use C and C++ libraries in your code.
  ::

  ::card{}
  #title
  Improved Performance
  #description
  This should be self explanatory, but cobalt24 will be faster than cobalt23.
  ::

  ::card{}
  #title
  Decorators
  #description
  Decorators will allow you to bind to signals using `@signal` like in python.

  ::card{}
  #title
  Pointers
  #description
  `&x` and `*x` will be added to cobalt24, allowing you to use pointers in your code this will also work with C(++) and the core.memory library.

  ::card{}
  #title
  Extended Standard Library
  #description
  cobalt24 will introduce a larger standard library and more built in features.

  ::card{}
  #title
  Linter
  #description
  A linter library will be in the standard library, allowing you to detect warnings and errors in your code before you run it.
::
