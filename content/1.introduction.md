# Introduction

Welcome to the documentation for the `@cobalt-lang/cobalt` language. This includes guides and tutorials to 
make you a master of the language.

## What is Cobalt?
Cobalt is based of Lua and is made to be faster, more powerful, and a more low-level features
than Lua. Cobalt can be interpreted, bytecoded.

Some of Cobalts features are:
- Type checker
- C-like syntax (based on Ljs)
- Classes
- Macros
- Enums
- Ultrafast AOT and interpreter
- Manual memory management (with garbage collection)
- And more!
***
