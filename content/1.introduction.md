# Introduction

Welcome to the documentation for the `@cobalt-lang/cobalt` language. This includes guides and tutorials to 
make you a master of the language.

## What is Cobalt?
Cobalt is based of Lua and LuaJIT and is made to be faster, more powerful, and a more low-level features
than Lua. Cobalt can be interpreted, bytecoded, or compiled to assembly and JIT. Some of Cobalts features
are:
- Type checker
- C-like syntax (based on Ljs)
- Classes
- Macros
- Enums
- Ultrafast JIT and interpreter
- Manual memory management (with garbage collection)
- And more!
***
# Example code:
```js filename="cobalt.js" {0} copy
// This is a comment
var a = 5
var b: number = "I am not a number!" // b will be nil
var c: number! = "I am not a number!" // will throw an error
var d = (1+1 == 2) ? "Yes" : "No" // d will be "Yes" because 1+1 == 2

if (b == c){
    print("b and c are equal")
} else {
    print("b and c are not equal")
}

// Memory access
var address = core.memory.alloc(1024) // Allocates 1024 bytes of memory
core.memory.free(address, 1024) // Frees the memory

enum("State", {
    "a": 1,
    "b": 2,
    "c": 3
})

if (jit){
    // Is this code being JIT compiled?
    print("Yes")
    jit.on() // Turn JIT on
    jit.off() // Turn JIT off
} else {
    print("No")
}

print(State.a) // 1

for (i = 1, 10){ // Numbers from 1-10 like in Lua
    print(i)
}

if (core.macros("__APPLE__"), "1"){ // Real C macros
    print("This is an Apple device")
} else {
    print("This is not an Apple device")
}
```
