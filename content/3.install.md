# Installation and Building
## Prebuilt Binaries
We do not have any prebuilt binaries yet, but we will soon.

## Dependencies
- [CMake](https://cmake.org/download/)
- [Git](https://git-scm.com/downloads)
- [Python 3](https://www.python.org/downloads/)
- [Make](https://www.gnu.org/software/make/) **NOTE:** Windows users will need to install [MinGW](https://www.mingw-w64.org) or [Cywin](https://www.cygwin.com) to use Make
- GCC (or any other C compiler)
## Building
### 1. Clone the repository
```bash
git clone https://github.com/cobalt-lang/cobalt.git
```
### 2. Change directory to the repository
```bash
cd cobalt
```
### 3. Identify your target platform:
- **Windows** - Continue this list
- **Posix** (Unix, macOS, Linux, etc.) - Continue this list
- **Xbox** - [Xbox Setup](/platforms)
- **PlayStation** - [PlayStation Setup](/platforms)
- **Nintendo** - [Nintendo Setup](/platforms)
- **Web** - [Web Setup](/platforms)
- **Android** - [Android Setup](/platforms)
- **iOS, vision/xrOS, watchOS, iPadOS, tvOS** - [Apple Setup](/platforms)
- **Other** - [Other Setup](/platforms)
### 4. Build the build folder
```bash
cmake .
```
### 5. Build the project
```bash
make # Makes cobalt and cobaltc
```
### 6. Optional JIT
```bash
make jit # Makes cobaltjit
```
### 7. Optional lua-cobalt (transpiler)
```bash
make lua-cobalt # Makes lua-cobalt
```
### 8. Install to your system
This step requires python, but python has many different names on different systems. Here are some common ones:
**python3:**
```bash
make setup # Installs cobalt and cobaltc
```
**py:** (Windows usually)
```bash
make setuppy # Installs cobalt and cobaltc
```
**python:**
```bash
make setup-py # Installs cobalt and cobaltc
```
Once you do any othese, the script will find the executables, and install them to your system, and build the standard library.
### 9. Testing
```bash
$ cobalt
```
That opens the shell.
