## Dependencies
- [CMake](https://cmake.org/download/)
- git

### 1. Clone   
```bash
git clone https://github.com/cobalt-lang/cobalt
```
### 2. Go into the src
```bash
cd cobalt
```
### 3. Build the build folder
> **Flags:**
> - -DFFI=ON (default: ON). Make Cobalt with an FFI (no dependencies but a x64, x86, or ARM CPU)
> - -DSDL=ON (default: OFF). Make Cobalt with SDL2 bindings (requires SDL, SDL_image, SDL_mixer, SDL_ttf)
> - -DSOCKET=ON (default: ON). Make Cobalt with cURL bindings (requires libcurl)
```bash
cmake -DSDL=OFF -DSOCKET=ON -DFFI=ON . # Makes cobalt, cobaltc, cobaltaot, cobaltpre, and minicobalt buildfiles
# with FFI & cURL but no SDL. (default config)
```
### 4. Build the binaries
```bash
make
```
### 5. lua-cobalt (optional)
```bash
make lua-cobalt # Makes lua-cobalt (requires Unix Makefiles)
```
### 6. Testing (optional)
```bash
$ make test
```
That will use the binaries to run 60 tests (20sec).

### 7. Install to your system
This step requires python, but python has many different names on different systems. Here are some common ones:
```bash
make install # Installs the binaries and include path.
```

Once you do any othese, the script will find the executables, and install them to your system, and build the standard library.
