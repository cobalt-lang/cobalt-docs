# Highlighting
> This is for the Syntax Highlighter for cobalt, available for VSCode.
## Installation
### From the VSCode Marketplace
1. Open the extensions sidebar on VSCode
2. Search for `Cobalt Syntax Highlighter`
3. Click install
### From the VSIX
1. Download the latest release
2. Open the extensions sidebar on VSCode
3. Click the three dots
4. Click `Install from VSIX`
5. Select the downloaded file
